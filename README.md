## xbps_packages
Collection of Void Linux style XBPS packages assembled for KLV 


- to download a package via terminal from github.com
  - right-click on the file and click copy link address


```
https://github.com/techrockedge/xbps_packages/blob/main/Conky_toggle_switch-1.0_1.noarch.xbps
```
- then bring it to the form below


```
wget https://github.com/techrockedge/xbps_packages/blob/main/Conky_toggle_switch-1.0_1.noarch.xbps?raw=true -O Conky_toggle_switch-1.0_1.noarch.xbps
```
